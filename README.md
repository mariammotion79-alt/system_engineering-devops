# 0x01. Shell Permissions

## Description
This project covers Linux file permissions and user management.
It contains Bash scripts that demonstrate how to manage users,
groups, and file permissions in a Linux environment.

## Learning Objectives
- How to change user with su
- How to check current user with whoami
- How to list user groups
- How to change file owner and group
- How to modify file permissions using chmod
- How to create files and directories with specific permissions

## Requirements
- All scripts are exactly 2 lines long
- All files start with #!/bin/bash
- All files are executable

## Files
| File | Description |
|------|-------------|
| 0-iam_betty | Switches current user to betty |
| 1-who_am_i | Prints effective username |
| 2-groups | Prints all groups of current user |
| 3-new_owner | Changes owner of hello to betty |
| 4-empty | Creates an empty file called hello |
| 5-execute | Adds execute permission to owner |
| 8-James_Bond | Sets permissions 007 on hello |
| 9-John_Doe | Sets permissions 753 on hello |
| 11-directories_permissions | Adds execute to all subdirectories |
| 12-directory_permissions | Creates my_dir with permissions 751 |
| 13-change_group | Changes group of hello to school |
